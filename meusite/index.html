Criatividade

A criatividade é ideal na hora de desenvolver um site, além do pedido do cliente o desenvolvedor usa seu talendo criativo para que o site seja bem visto na web, como também acessível para o público em geral. Um design bonito e acessível é o que todo mundo procura!

Responsividade

Responsividade é a forma em que o site se adapta a cada dispositivo, sendo ele um computador de alta resolução ou mesmo um smartphone antigo com uma largura de 320px. Mesmo sem uso de Bibliotecas consigo criar a responsividade apenas com Media Queries e Javascript, um código único para seu site e do seu jeito!

SEO

SEO é como os motores de buscar vêm seu site na web, existem técnicas que melhoram a forma que seu site é visto por esses dispositivos, são eles Yahoo, Google, ou Bing. Irei aplicar as melhores técnicas de SEO para que seu site apareça na primeira página do google. Pesquise meu portfólio por exemplo. Uma boa prática de SEO é fundamental!

Um pouco sobre mim

As tecnologias que utilizo são HTML5, CSS3, JAVASCRIPT, JQUERY, e BOOTSTRAP (apesar de gostar de criar meus sites responsivos sem frameworks).
